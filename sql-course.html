🧮 SQL for Data Science

Learn how to query, analyze, and manipulate data using Structured Query Language (SQL). This course covers everything from foundational SELECT statements to analytical functions used in data science.

📘 What You'll Learn

  • Basic SQL syntax and SELECT queries
  • Filtering, sorting, grouping, and joins
  • Subqueries, CTEs, and window functions
  • Using SQL for exploratory data analysis (EDA)
  • SQL case studies and datasets

📂 Course Format

  • 💻 Hands-on SQL notebooks
  • 📊 Practice queries on public datasets
  • 📝 Final project: SQL reporting dashboard

🛠️ Tools You'll Use

  • PostgreSQL or MySQL (browser-based or desktop)
  • SQL IDEs: DB Fiddle, DBeaver, PgAdmin

🎁 Bonus Materials

  • SQL syntax cheat sheet
  • Top 50 SQL interview questions
