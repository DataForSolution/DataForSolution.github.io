Power BI Fundamentals

Learn how to transform raw data into interactive dashboards and data visualizations using Microsoft Power BI. This course is ideal for beginners and business professionals looking to strengthen their data analysis and reporting skills.

What You Will Learn

  • Connecting to various data sources (Excel, SQL, Web APIs)
  • Cleaning, transforming, and modeling data using Power Query
  • Designing effective visuals such as bar charts, KPIs, and slicers
  • Publishing, collaborating, and sharing reports via Power BI Service
  • Writing DAX expressions for calculated columns and advanced measures

Course Format

  • Pre-recorded video lectures and walkthroughs
  • Guided practice datasets and exercises
  • Capstone project: Build a dynamic sales dashboard

Tools and Technologies

  • Power BI Desktop (Windows version)
  • Power Query Editor
  • DAX (Data Analysis Expressions)

  • DAX function cheat sheet (PDF)
