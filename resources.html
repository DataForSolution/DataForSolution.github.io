Resources for Data Science Enthusiasts

Explore a curated collection of tools, datasets, tutorials, and guides to enhance your data science journey.

Free Datasets

Practice your skills with these free and publicly available datasets.

  • Data-artikelen | Sargasso - Data journalism and articles.
  • The Guardian Datablog - Data journalism and visualization from The Guardian.
  • Knoema - A comprehensive data marketplace.
  • Amazon Public Datasets - Large public datasets available on AWS.
  • Socrata - Data solutions for governments and organizations.
  • Data Publica - Business-oriented datasets.
  • Archive-It - Web archiving services for institutions.
  • Google Public Data Explorer - Visualize and analyze public data.
  • Zanran - Search for numerical data.
  • Quandl - Search and download economic and financial datasets.
  • IMF Data and Statistics - Data from the International Monetary Fund.
  • The World Bank - Global development data.
  • OECD.Stat - Data from the Organisation for Economic Co-operation and Development.
  • UN Data - Global statistics and datasets from the United Nations.
  • European Environment Agency - Environmental datasets and maps.
  • London DataStore - Open data from London.
  • Seattle Open Data - Public datasets from Seattle.
  • Chicago Data Portal - Open data from Chicago.
  • Austin Data Portal - Public data for Austin.
  • San Francisco Data - Open datasets from San Francisco.
  • Kaggle Competitions - Data science and machine learning challenges.
  • Causality Workbench - Challenges focused on causal inference and machine learning.
  • Institute for Health Metrics and Evaluation (IHME) - Health-related datasets.
  • Gapminder - Statistics for global development.
  • UCI Machine Learning Repository - Comprehensive machine learning datasets.
  • DataHub.io - Share and publish datasets.
  • Data.gov - The official U.S. open data portal.
  • Data.gov.uk - The official UK open data portal.
  • Data.gov.au - Open data portal for Australia.
  • Open Data Canada - Canada’s open government portal.
  • Kaggle Datasets - A hub of datasets for all levels of data science projects.
  • UCI Machine Learning Repository - A comprehensive archive for machine learning datasets.
  • Data.world - A platform for open data sharing and discovery.
  • Awesome Public Datasets - A collection of free datasets hosted on GitHub.

    • Open-Source Tools

    Access powerful tools and libraries to accelerate your projects.

    • Python - A versatile programming language for data science and machine learning.
    • Pandas - A library for data manipulation and analysis.
    • Plotly - Interactive graphing and dashboard creation.
    • Scikit-learn - A machine learning library for Python.
    • Jupyter Notebooks - An open-source tool for interactive computing.

    Tutorials and Guides

    Learn from industry experts and kickstart your data science career.

    • Coursera - Online courses from top universities and companies.
    • Kaggle Learn - Free micro-courses on data science and machine learning.
    • Towards Data Science - Blog articles and tutorials by data science professionals.
    • Google ML Crash Course - A quick introduction to machine learning by Google.

    Code Snippets and Templates

    Jumpstart your projects with pre-built templates and reusable code.

    • GitHub - Explore open-source repositories for inspiration.
    • LLmachine Code Snippets - Custom templates and scripts for common tasks.
    • GitHub Gists - Share and discover reusable code snippets.
